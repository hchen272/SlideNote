# Slide Note（PPT笔记）

> 一个桌面便签应用：永远置顶、可吸附到屏幕边缘如书签、支持 Markdown 与富文本编辑。

[English](README.md)

---

## 为什么做这个

现有的电脑便签工具有两个痛点：

1. **不能真正置顶。** Windows 自带便签等，一切换窗口就被挡住。读 Slides、看视频、查资料时，笔记应该**持续可见**，而不是藏在窗口背后。

2. **占地方或难切换。** 传统笔记软件要么占屏幕空间，要么需要反复 Alt-Tab。理想方案是让笔记**活在屏幕边缘**——随时可用，从不碍事。

**Slide Note** 解决了这两个问题：始终置顶（全屏应用除外），一键收缩成屏幕边缘的彩色小标签，再点一下恢复完整窗口，笔记全部保留。

**核心场景：** 边看 Slides / PDF / 视频课边做笔记，无需来回切窗口。

---

## 功能

- **永远置顶** — 悬浮在其他窗口之上（全屏应用自动让位）
- **屏幕边吸附** — 收起为贴边彩色标签，可拖拽移动
- **4 套主题** — 赛博朋克、自然、中世纪、极简
- **Markdown + 富文本** — 双编辑模式，一键互转
- **TODO 清单** — 编辑/预览模式均可点击勾选
- **大纲导航** — 标题层级树，点击跳转
- **笔记文件夹** — 标签式文件夹，彩色标记，多选批量操作
- **链接** — 插入/编辑链接；Ctrl+点击在浏览器打开
- **LaTeX 公式** — KaTeX 渲染，行内 `$...$` 和块级 `$$...$$`
- **表格** — 自定义行列数插入；GFM 表格支持
- **便签管理** — 新建、删除、搜索、排序（修改时间/创建时间/字数/首字母）
- **字体自定义** — 每条便签独立设置字号、粗细、颜色
- **独立文件存储** — 每条便签一个 JSON 文件，便于备份和 Git 管理
- **自定义存储路径** — 设置中修改，自动迁移
- **中英双语** — 设置中切换

## 环境配置

### 前提

- [Node.js](https://nodejs.org/) v18+（推荐 v24）
- npm

### 安装运行

```bash
git clone https://github.com/hchen272/SlideNote.git
cd SlideNote
npm install
npm run electron:dev
```

### 打包安装包

```bash
npm run electron:build
# 输出：release/Slide Note Setup.exe
```

## 操作指南

| 操作 | 方式 |
|------|------|
| **新建便签** | 侧边栏点击 **+ 新建便签** |
| **新建文件夹** | 点击 📁+ 选择颜色 |
| **编辑 / 预览** | 工具栏 👁️ 按钮 |
| **切换主题** | 工具栏主题下拉（🌆 / 🌿 / 🏰 / ⬛） |
| **吸附到屏幕边** | 标题栏 ◀ 按钮 或 `Ctrl+Shift+D` |
| **从边缘展开** | 点击屏幕边缘彩色标签 |
| **大纲面板** | 标题栏 ☷ 按钮 |
| **搜索便签** | 侧边栏搜索框 |
| **排序** | 侧边栏下拉（修改时间/创建时间/字数/首字母） |
| **多选模式** | 排序栏 ☐ 按钮 |
| **右键便签** | 删除、添加到文件夹、移出文件夹 |
| **打开设置** | 侧边栏底部 ⚙ 按钮 |

### Markdown 速查

```markdown
# 一级标题
## 二级标题
**粗体**  *斜体*  `代码`

- 无序列表
- [ ] 待办事项
- [x] 已完成事项

> 引用

[链接文字](url)
```

## 技术栈

- **Electron** — 桌面框架
- **React 18** + **TypeScript** — 前端

- **Slate.js** — 富文本编辑
- **marked** — Markdown 渲染
- **KaTeX** — LaTeX 公式渲染
- **electron-store** — 配置持久化
