# Sticky Notes（便签）

> 一个桌面便签应用：永远置顶、可吸附到屏幕边缘如书签、支持 Markdown 与多主题。

[English](README.md)

---

## 为什么做这个

现有的电脑便签工具有两个痛点：

1. **不能真正置顶。** Windows 自带便签、macOS 便条等，一切换窗口就被挡住。读 Slides、看视频、查资料时，笔记应该**持续可见**，而不是藏在窗口背后。

2. **占地方或难切换。** 传统笔记软件要么占屏幕空间，要么需要反复 Alt-Tab。理想方案是让笔记**活在屏幕边缘**——随时可用，从不碍事。

**Sticky Notes** 解决了这两个问题：始终置顶（全屏应用除外），一键收缩成屏幕边缘的彩色小标签，再点一下恢复完整窗口，笔记全部保留。

**核心场景：** 边看 Slides / PDF / 视频课边做笔记，无需来回切窗口。

---

## 功能

- **永远置顶** — 悬浮在其他窗口之上（全屏应用自动让位）
- **屏幕边吸附** — 收起为贴边彩色标签，可拖拽移动位置
- **3 套主题** — 赛博朋克、自然、中世纪
- **Markdown 编辑** — 支持 Markdown 语法 + 实时预览
- **TODO 清单** — 可勾选/划掉的待办（`- [ ]` / `- [x]`）
- **便签管理** — 新建、删除、搜索，按创建时间/修改时间/字数排序
- **字体自定义** — 每条便签可独立设置字号、粗细、颜色
- **数据持久化** — 自动保存，支持在设置中自定义存储路径

## 环境配置

### 前提

- [Node.js](https://nodejs.org/) v18+（推荐 v24）
- npm（随 Node.js 安装）

### 安装运行

```bash
# 克隆仓库
git clone <your-repo-url>
cd sticky-notes

# 安装依赖
npm install

# 开发模式启动
npm run electron:dev
```

`.gitignore` 仅忽略 `node_modules/` 和 `dist-electron/`，因此克隆后执行上述命令即可运行。

### 打包发布

```bash
npm run electron:build
```

打包后的应用在 `release/` 目录下。

## 操作指南

| 操作 | 方式 |
|------|------|
| **新建便签** | 侧边栏点击 **+ 新建便签** |
| **编辑 / 预览切换** | 工具栏 👁️ 按钮 |
| **切换主题** | 工具栏主题图标（🌆 赛博朋克 / 🌿 自然 / 🏰 中世纪） |
| **吸附到屏幕边** | 标题栏 ◀ 按钮 或 `Ctrl+Shift+D` |
| **从边缘展开** | 点击屏幕边缘的彩色标签 |
| **移动吸附标签** | 拖拽标签沿屏幕边缘上下移动 |
| **搜索便签** | 侧边栏搜索框 |
| **排序** | 侧边栏下拉（修改时间/创建时间/字数） |
| **插入 Markdown** | 工具栏 H / **B** / *I* / ☑ 等按钮 |
| **修改字体** | 字号下拉 / B 加粗切换 / A 颜色选取 |
| **打开设置** | 侧边栏底部 ⚙ 按钮 |

### Markdown 速查

```markdown
# 一级标题
## 二级标题
**粗体**  *斜体*  `代码`

- 无序列表
- [ ] 待办事项
- [x] 已完成事项

> 引用

[链接文字](url)
```

## 自定义存储路径

默认数据保存在：

```
C:\Users\<用户名>\AppData\Roaming\sticky-notes\config.json
```

修改方式：侧边栏底部 ⚙ → 输入新文件夹路径 → 保存。下次启动生效。
